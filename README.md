<div align="center"><h1>Want to try it?</h1></div>


1. Clone the repository
```sh
git clone https://github.com/matias-llanquiman/React-todo-list.git
```
2. Install all dependencies
```sh
npm install
```
3. Run the local dev server
```sh
npm run dev
```
